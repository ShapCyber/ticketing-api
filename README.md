# Clone repository
* Clone the repo: `git clone git@github.com:kings-cam/ticketing-api tickets`
* `cd` to the repo: `cd tickets`

# Install dependencies

* `go get github.com/labstack/echo`

# Compile

`go build cmd/api/server.go`

# Run

`./server`
